# BrainBattle - Mobile Applications Project

BrainBattle is an Android mobile application inspired by the Serbian TV quiz *Slagalica*.  
The application allows users to access different quiz mini-games, player profiles, statistics, and notifications through a unified game menu.

## Technologies

- Java
- Android Studio
- XML Layouts
- Android SDK

## How to Run the Application

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Open the project in Android Studio.

3. Wait for Gradle sync to finish.

4. Connect an Android device or start an emulator.

5. Run the application using the green **Run** button.

## Application Flow

- Splash screen
- Login / Registration
- Home screen (main game menu)
- Access to:
  - Profile
  - Quiz
  - Matching
  - Associations
  - Skocko
  - Step by Step
  - My Number
  - Notification Center

## Notes

- Current version contains GUI and navigation flow for KT1.
- Authentication and backend functionality are not yet implemented.
- Buttons currently navigate between screens without validation logic.
